# 🚀 MotoAI – AI Virtual Assistant for Any Website

## 🌟 Overview
MotoAI is an AI-powered SaaS platform that allows website owners to embed a fully customizable, voice-enabled AI assistant into any website with just one line of code. The assistant can answer visitor questions, speak responses, and navigate users across website pages using voice commands.

---

## ✨ Features

- 🤖 AI-powered conversations (Google Gemini)
- 🎤 Voice input & voice output
- 🌐 Website page navigation via voice
- 🎨 4 Premium themes (Dark, Light, Glass, Neon)
- 🔑 Google Authentication
- 💳 Razorpay Subscription
- 📊 Free & Pro Plans
- 📦 One-line JavaScript embed
- 🔒 JWT Authentication
- ⚡ Fast React + Vite frontend
